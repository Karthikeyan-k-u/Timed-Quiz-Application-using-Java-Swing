# Advanced Timed Quiz Application (Java Swing)

## 📌 Overview

This project is a GUI-based Quiz Application developed using Java Swing. It allows users to answer multiple-choice questions within a limited time. The application automatically moves to the next question when time expires.

---

## 🚀 Features

* Interactive GUI using Java Swing
* Multiple-choice questions (MCQs)
* Countdown timer (5 seconds per question)
* Automatic next question when time ends
* Displays final score at the end
* Clears previous answers for better user experience

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* Multithreading (Timer implementation)

---

## 📂 Project Structure

* `Mini4.java` → Main class containing GUI, logic, and timer

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Mini4.java
   ```
3. Run the program:

   ```bash
   java Mini4
   ```

---

## 📊 How It Works

* The application displays one question at a time.
* Each question has 4 options.
* A timer of 5 seconds starts for each question.
* User selects an answer and clicks **Next**.
* If time runs out, it automatically moves forward.
* The score is calculated based on correct answers.
* Final score is displayed at the end.
